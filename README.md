# E-Commerce SQL Database Analysis Project

## Project Overview

This project is a beginner-level E-Commerce database project developed using MySQL.

## Database Structure

The database contains five main tables:

- Customers
- Products
- Orders
- Order Items
- Payments

## SQL Concepts Used

- Database creation
- Table creation
- Primary Keys
- Foreign Keys
- Auto Increment
- Data insertion
- SQL JOIN operations
- Aggregate functions (SUM, COUNT)

## Analysis Performed

The project includes SQL queries to analyse:

- Total revenue generated
- Product-wise revenue performance
- Delivered orders analysis
- Cancelled orders count

## Project Files

### 01_database_schema.sql

Creates the complete database structure.

### 02_insert_sample_data.sql

Contains sample e-commerce data.

### 03_analysis_queries.sql

Contains SQL queries used for data analysis.

## Tools Used

- MySQL
- MySQL Workbench
- GitHub

## Key Learning Outcomes

- Designing relational databases
- Creating table relationships
- Writing SQL queries
- Using joins for analysis
